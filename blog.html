Question-1: What are the different ways to select an element in the DOM?

Answer : There are six ways to write (DOM),
example :
getElementById("id") , getElementsByClassName("className") , getElementsByTagName("tagName") , querySelector("selector") , querySelectorAll("selector") ,getElementsByName('nameAttribute')

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : textContent property returns the text content of the node. innerHTML property returns the HTML content of the node. innerText property returns the text content of the node.

Question-3: What is event delegation in the DOM?

Answer : Event delegation is a technique in JavaScript that allows you to attach event listeners to a parent element instead of attaching them to each individual child element.

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling is a concept in JavaScript where events are propagated from the innermost element to the outermost element.
